- AuditMiddleware (`internal/s3api/audit.go`) wraps state-changing requests; GET/HEAD/OPTIONS skipped. Add publishers via `Publisher AuditPublisher` field after construction (publish only on `EnqueueAudit` success).
- adminapi routes register via `mux.HandleFunc("VERB /admin/v1/...", handler)` in `Server.routes()` (Go 1.22 mux). Trailing-wildcard `{key...}` must be the last segment.
- Prometheus metrics: declare in `internal/metrics/metrics.go` `var ()` block; add to `Register()`; expose adapter type implementing the consumer-package's `MetricsSink`-style interface. Keeps prometheus out of leaf packages.
- adminapi unit tests bypass `authMiddleware` by calling `s.routes().ServeHTTP` directly with `withAuditAuthCtx` ctx wrapper.
- SSE handlers must `flusher.Flush()` after each frame and write `:keep-alive\n\n` pings on a ticker; race-test with `t.Context()` and ensure subscriber registration before publishing in tests.
- Web sidebar nav is sectioned (`web/src/components/layout/nav.ts` exports `NavSection[]`). Add Phase-N pages by appending to an existing section or pushing a new `{label, items}` block; `SidebarNav.tsx` renders headers automatically and hides them in collapsed mode.
- SSE consumers in the UI use `web/src/hooks/useEventSource.ts`. Hook owns reconnect/backoff (1sâ†’30s) and exposes `state` + `retryInSeconds` for status banners. Filter changes that need to re-subscribe go through the URL prop â€” do NOT add filter state inside the hook.

- Adding a column to `audit_log`: push DDL into `cassandra/schema.go::alterStatements`, then update INSERT + every SELECT (Enqueue/ListAuditFiltered/ReadAuditPartition/ListAudit). TiKV `auditRow` JSON tags accept a new `,omitempty` field with no migration. `meta.AuditEvent` is the lockstep struct.
