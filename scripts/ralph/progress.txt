## Codebase Patterns
- Optional capability surfaces (RangeScanStore, HealthProbe) hang off the
  base meta.Store interface. Backends implement them via type-assertion at
  the call site (e.g. internal/adminapi handler) — keeps the base interface
  small while allowing per-backend differentiation. Add a compile-time
  `var _ meta.<Iface> = (*Store)(nil)` assertion in each backend.
- Cassandra system-table queries use Consistency(gocql.LocalOne) for
  cluster-introspection (not LOCAL_QUORUM), since system.* tables are
  per-node and quorum reads make no sense.
- Admin endpoints register in routes() under server.go and live in their
  own file (storage.go, cluster.go, ...). Tests for those endpoints live
  next to them as <name>_test.go and use newTestServer() to drive
  s.routes().ServeHTTP(rr, req) directly without auth.
