## Codebase Patterns

- web/ is a NEW source tree at the repo root (sibling of cmd/, internal/, docs/).
  React 18 + Vite 5 + TypeScript 5. Phase 1 stack will add Tailwind 3 + shadcn/ui +
  TanStack Query 5 + recharts 2 + react-router-dom 6 + Playwright 1. Package
  manager: pnpm 9 (NOT npm/yarn — smaller node_modules, reproducible install).
  Lockfile checked in: web/pnpm-lock.yaml.
- go:embed pattern: ROOT package `strata` (not cmd/strata/) owns console.go +
  console_test.go. Declares `//go:embed all:web/dist` + `var consoleFS embed.FS`
  and exposes `ConsoleHandler()` with SPA fallback (any /console/<path> not matching
  a static file falls back to index.html so client-side routes survive direct
  navigation / refresh). The `all:` prefix matters — without it Vite's .vite/-
  prefixed assets are silently dropped. cmd/strata-gateway imports this as
  `strataconsole "github.com/danchupin/strata"`.
- /console/ + /admin/v1/* are reserved path prefixes on the gateway port. Both
  register ahead of the catch-all / S3 router in cmd/strata-gateway/main.go (no
  internal/serverapp package — wiring is inline in main.go). reservedBucketNames
  in internal/s3api/validate.go already includes 'console', 'admin', 'metrics' so
  path-style addressing cannot shadow gateway-internal routes.
- /admin/v1/* is the JSON HTTP contract between UI and gateway. Versioned (v1);
  future-breaking changes ship under /admin/v2/*. OpenAPI 3.1 spec lives at
  internal/adminapi/openapi.yaml (US-003).
- Auth: session cookie strata_session=<JWT> with HS256, 24h Max-Age, HttpOnly +
  Secure + SameSite=Strict + Path=/admin. JWT secret read from
  STRATA_CONSOLE_JWT_SECRET (random hex if unset, with WARN — production
  deployments MUST set it explicitly to survive restarts without invalidating
  sessions). Auth middleware on /admin/v1/* falls back to existing SigV4 path for
  clients without the cookie (curl, programmatic admin clients).
- TanStack Query 5 global QueryClient: staleTime 5_000, refetchInterval 5_000,
  refetchOnWindowFocus true, retry 1. web/src/api/client.ts exposes typed
  wrappers — pages call useQuery({queryKey, queryFn}); polling is automatic.
- Heartbeat shape: cluster_nodes table with default_time_to_live=30 on Cassandra
  (alterStatements add), s/cluster_nodes/<node_id> + background sweeper on TiKV,
  in-process map with timestamp expiry on memory. Lockstep across all three meta
  backends. Each Strata replica writes its own row every 10 s.
- Status derivation (cluster overview): healthy if all nodes report /readyz 200
  in last 30 s; degraded if ≥1 fail but quorum holds; unhealthy if quorum lost.
- Polling cadence per page: 5 s for ranges ≤6 h, 30 s for 24h+, 5 min for 7d
  (per-query refetchInterval). Mind chart re-render cost on longer ranges.
- Bundle target: ≤500 KiB gzipped initial. Use React.lazy + Suspense for
  non-critical routes (Buckets, Bucket Detail, Metrics get code-split). Verify
  via `vite build --report` before US-012 close. Current bundle: 46 KiB gzipped.
- recharts 2.x for charts (smaller than chart.js, declarative). Avoid Apache
  ECharts / Plotly — bundle bloat.
- shadcn/ui CLI copies components into web/src/components/ui/ as source — NOT a
  dependency. We own them; tweak freely, no version pin to worry about.
- Dark mode: class='dark' on <html>; CSS variables in globals.css switch values
  per theme. Toggle persists to localStorage['strata.theme'] = 'light' | 'dark' |
  'system'.
- Playwright e2e: chromium-only on CI for speed. webServer config runs
  `make run-memory` so e2e does not need Cassandra / RADOS up.
  STRATA_STATIC_CREDENTIALS=test:test:owner seeds login creds.
- Gateway dev port: make run-memory uses :9999 (NOT :9000 as some PRD ACs
  suggest). Update Playwright baseURL to http://localhost:9999 to match the
  Makefile.
- Makefile: top-level `build` depends on `web-build`. `make web-typecheck` and
  `make web-clean` available. CI ci.yml has a web-build job that produces the
  web/dist artifact downloaded by go jobs (go test/build needs it for the embed).
- ROADMAP maintenance rule (CLAUDE.md): every commit closing a ROADMAP item
  flips the bullet to ~~strikethrough~~ Done format in the same commit, with
  closing SHA inline (or `(commit pending)` with a follow-up SHA edit). New gaps:
  add a fresh entry under the right severity section in the same commit.
- Phase boundaries (read these BEFORE adding scope creep):
  - Phase 1 (THIS cycle, prd-web-ui-foundation.md): read-only console.
    No write actions.
  - Phase 2 (prd-web-ui-admin.md): bucket admin, IAM, object actions, multipart
    watchdog, audit viewer, settings.
  - Phase 3 (prd-web-ui-debug.md): hot heatmaps, slow queries, OTel trace
    browser, SSE live tail, per-node drilldown.
  If a story tempts you toward write actions or debug tooling, STOP and re-scope.
  Phase 1 ships first; the others land in their own cycles.
- No metadata schema changes outside the new cluster_nodes heartbeat table. No
  data backend changes. No SSE / WebSocket — polling 5 s is sufficient for
  Phase 1.
- Tests for the cycle: Go side via `go test ./internal/adminapi/...`, Playwright
  via `cd web && pnpm exec playwright test`. CI has e2e-ui job that runs both.

- **Learnings for future iterations:**
  - The gateway binary is cmd/strata-gateway/, not cmd/strata/. The PRD AC text
    "cmd/strata/console.go" was descriptive shorthand — actual location is the
    REPO ROOT (package strata) so go:embed can see web/dist. Don't try to "fix"
    this by moving files into cmd/strata-gateway/.

    one). Mux wiring is inline in cmd/strata-gateway/main.go. Future stories that
    need to register handlers should add them in main.go alongside the existing
    /metrics, /console/ handlers — same pattern.
  - `make run-memory` listens on :9999 not :9000. Playwright config baseURL must
    match (http://localhost:9999). aws-cli + curl examples in docs/ui.md should
    use 9999 too.
  - When the ralph wrapper rotates a cycle, the on-disk prd.json may carry stale
    content from the previous cycle (s3-compatibility) until the first iteration
    of the new cycle restores it. Check branchName in prd.json before picking a
    story; if it doesn't match scripts/ralph/.last-branch, restore the PRD body
    first.
  - The bundle is 46 KiB gzipped at the bare-skeleton stage. Tailwind + shadcn
    primitives + TanStack Query + recharts + react-router will add weight; budget
    ~150–250 KiB for them combined. Use route-level React.lazy + Suspense to keep
    the initial paint light (US-005 / US-009 / US-010 / US-011 are good code-split
    boundaries).
