**Learnings for future iterations:**
- meta.Bucket gains 3 fields → TiKV codec is JSON marshal, so persistence is free.
  Cassandra needs explicit ALTER + Get/Set column selection. Memory backend stores
  meta.Bucket directly.
- LWT (`UPDATE ... IF EXISTS`) on Cassandra is required for the signing-key trio
  to participate in the same Paxos lineage as CreateBucket — mirrors the
  SetBucketVersioning lesson (CLAUDE.md captures it).
- `subtle.ConstantTimeCopy(1, dst, src)` is the canonical zero-out for sensitive
  byte slices on eviction.
- HTTP middleware deny handler `s3api.WriteAuthDenied` switches on `err.Error()`
  string equality for legacy auth errors AND `errors.Is` for new typed sentinels —
  mixing both is the established pattern.
- Per-bucket signing key shape: SigV4 derives signing key from
  `hex(DEK)`-as-secret. Operators sign with the same hex on the client side; AWS
  CLI accepts arbitrary `aws_secret_access_key` so this is wire-compatible.
- Testcontainers v0.41.0 LocalStack image: pin `localstack/localstack:3.8` —
  `:latest` flips behavior periodically. Health endpoint
  `/_localstack/health` returns 200 once `SERVICES=kms` is up.
