  - admin.Run vs admin.RunWith — split entrypoint into writer-injected variant (RunWith) so the top-level dispatcher tests in `cmd/strata` package can capture stdout/stderr via bytes.Buffer. The bare `Run(args)` wraps RunWith with os.Stdout/Stderr for production use. Cleaner than os.Pipe acrobatics in tests.

  - When converting `package main` → library package: the legacy `os.Exit(2)` shape inside `main()` cannot survive the move because tests can't catch `os.Exit`. Replace with `return ErrUsage` / `return err`; the caller in cmd/strata maps any non-nil to exit 2. Preserves CLI exit-code contract without locking tests out of the entrypoint.
