### Learnings for future iterations
- **Config substruct pattern**: every new env knob lives on a substruct under `Config`, with a koanf tag, a default in `config.defaults()`, an envMap entry, validation/clamp in `config.validate()`, a row in `deploy/strata.toml.example`, AND a row in `docs/site/content/reference/env-vars.md`. The drift-lint test `internal/config/env_toml_parity_test.go` catches misses on any of the four — running it post-change is faster than searching for missed sites.
- **Zero-as-disabled is a Go HTTP idiom**: `net/http.Server` treats zero-valued `ReadTimeout`/`WriteTimeout`/`IdleTimeout`/`ReadHeaderTimeout`/`MaxHeaderBytes` as "no timeout / default". US-001 mirrors that semantic for the operator UX (set `0s` → opt out) without inventing a sentinel value.
- **Negative duration parsing**: `time.ParseDuration("-1s")` succeeds with a negative `time.Duration`. Validation MUST explicitly reject negatives because koanf will accept them silently. Mirror this for every new duration env added in later stories.
- **Test pattern for env-driven config**: use `t.Setenv("STRATA_*", "...")` then call `config.Load()`. `t.Setenv` auto-cleans at test end so the helper composes cleanly with subtests. `STRATA_HTTP_WRITE_TIMEOUT=0s` round-trips through koanf as `time.Duration(0)` — no special-casing needed for the "explicit zero" path.
- **Slowloris chaos test shape**: spawn a real `http.Server` via `net.Listen("tcp", "127.0.0.1:0")` + `srv.Serve(ln)`, dial raw with `net.Dial`, dribble bytes, then `io.ReadAll` with `SetReadDeadline`. `io.ReadAll` returns `nil` on EOF (Go convention), surfaces RST as a non-timeout error, and only returns a `net.Error.Timeout()=true` when the server kept the socket open past the slack window. The deadline branch is the only failure mode.
- **`internal/data/rados/cephimpl/backend.go` build-tag diagnostic** (`OperationNoFlag` type mismatch) is unrelated to the default build — it surfaces only under `-tags ceph`. Default `go vet ./...` and `go build ./...` ignore it; CI's e2e job catches the tagged build separately.
- **ROADMAP "Cycle A" entry pattern**: NEW open entry inserted at the top of `## Correctness & consistency` in US-001; US-010 flips it to `~~Done~~` shape with the closing-SHA backfill on a follow-up commit. Mirrors the 12-prior-cycle close protocol — do NOT skip the open entry.
