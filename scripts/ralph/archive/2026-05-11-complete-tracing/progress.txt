  - manifestrewriter.Worker.Run is one-shot (returns Stats) — wrapping Run (not RunOnce, which doesn't exist) is the right injection point. The supervisor's manifestRewriterRunner loop in cmd/strata/workers/manifest_rewriter.go calls Run periodically — each call gets its own iteration span.
  - inventory test file is in external `inventory_test` package; the trace test goes alongside in that same `inventory_test` package, not `package inventory`. Same applies to any future per-worker trace test where the existing _test.go uses `package <name>_test`.
  - tracetest.InMemoryExporter + sdktrace.WithSyncer pattern reused across all 8 worker trace tests — single shared shape, no fancy mock tracer required. Helper `hasAttr(attrs, key, val)` is duplicated per package because exporting it from a common test util would force a new test package; copy-paste 6 lines per file is the lightest option.
