# Strata

Strata is a Go-based, S3-compatible object gateway designed as a drop-in
replacement for Ceph RGW. Metadata lives in Cassandra (or ScyllaDB / TiKV);
data lives in RADOS as 4 MiB chunks. The gateway speaks S3 over HTTP.

This documentation site is a placeholder scaffold (US-001 of the docs-site
cycle). Full landing page content lands in US-003; section pages follow in
US-002 onwards.
