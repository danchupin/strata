**Naming convention 2A**: `STRATA_<SECTION>_<KNOB>` → `<section>.<knob>`
(lowercase). First underscore splits section; remaining underscores stay
snake_case in the knob. Multi-word sections (e.g. `usage_rollup`,
`manifest_rewriter`, `audit_export`, `access_log`, `bucket_stats`, `quota_reconcile`)
MUST be declared explicitly in envMap so the wired TOML key matches the
canonical section name — the heuristic shown in the **Expected TOML key**
column below is the naive first-underscore split and WILL be wrong for
multi-word sections. The actual envMap entry is the source of truth.
