# Strata — Production-Readiness QA Report

**Cycle:** `ralph/qa-production-readiness` · **Started:** 2026-05-30 · **Verdict:** **PENDING** (locked until US-013)

This is the living evidence document for the QA production-readiness cycle. The
release go/no-go decision is made against the contents of this file, not vibes.
It is updated per-story; the final verdict is set in US-013.

Scope backends: **TiKV + memory** (canonical CI default), **RADOS/ceph**,
**S3-compatible (MinIO)**. Cassandra stays first-class in code but is gated off
the per-PR CI gate (see `.github/workflows/ci.yml::integration`).

**Source of truth = CI on Linux** (FR-4). Many failure modes (cgo runtime,
darwin-only `gosigar` sysctl, race timing) do not reproduce on a local
macOS/lima box. Numbers below tagged `CI (local env-blocked)` are measured on
the Linux CI coverage job, not the author's laptop.
