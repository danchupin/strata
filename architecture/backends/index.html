            Backends#

            Strata ships first-class production backends for both layers.

            LayerBackends
            MetadataCassandra (default), ScyllaDB (CQL drop-in), TiKV (raw KV with native ordered scan)
            DataRADOS (default, 4 MiB chunks), S3-over-S3 (any S3-compatible upstream), memory (tests only)
